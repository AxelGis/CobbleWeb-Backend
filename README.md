## Installation
### Create .env like env.example

`
.env.dev
.env.prod
`

## Running docker
```bash
# development
$ yarn run docker:dev

# production
$ yarn run docker:prod
```

## Running node

```bash
# install
$ yarn install

# development
$ yarn run start

# production mode
$ yarn run build
$ yarn run start:prod
```

## Test

```bash
$ yarn run test
```
